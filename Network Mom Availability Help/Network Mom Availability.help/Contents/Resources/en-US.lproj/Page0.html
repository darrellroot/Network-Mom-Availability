Network Mom Availability is a network and host monitoring application for IT professionals. It has the following features:

  • Group devices to be monitored into maps.

  • Associate each map with alert notification recipients and report notification recipients. In the current version all notifications use email delivery. Most cell providers have email->SMS gateways allowing short emails to result in text messages.

  • Ping hosts or network devices with an IPv4 ICMP ping.

  • Ping hosts or network devices with an IPv6 ICMP ping.

  • Measure and graph the “availability” of hosts (what percentage of pings they responded to).

  • Measure and graph the latency of hosts (how many milliseconds did it take to receive a ping reply).

  • Generate email alerts when devices change stage between “responsive” versus “unresponsive”.

  • Generate daily, weekly, and monthly email availability reports.

  • Automatically save data daily and before each (graceful) application exit.

  • Manually save data when requested (recommended before and after significant configuration changes).
