Availability monitor states

Network Mom Availability “tests” each monitor (with an IPv4 ping or an IPv6 ping) every 30 seconds. Availability monitors have the following states:

  • Gray: Never online. This monitor has not responded to a ping since the last Network Mom Availability restart or since the monitor was added to monitoring. It is possible this monitor is misconfigured, or that the target host blocks ICMP pings. If all your monitors turn Gray, your license may have expired.

  • Green: Online. This monitor passed recent tests and is responsive to ping. Green is good! That’s on the CCNA exam ;-)

  • Yellow: This monitor failed a test recently.

  • Orange: This monitor failed two tests recently.

  • Red: This monitor failed the three most recent tests.

Each time a ping is successful, the monitor transitions between states as follows:

Red -> Orange -> Yellow -> Green

For newly created monitors, or after a restart, the initial transition is as follows:

Gray -> Orange -> Yellow -> Green

When a target fails to respond to a ping within 30 seconds, the monitor transitions states as follows:

Green -> Yellow -> Orange -> Red
