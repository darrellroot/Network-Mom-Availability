Notifications

Network Mom Availability sends 2 types of notifications by email:

  • Alert: This is a short but frequent email sent when devices go down or recover. These are designed to be short enough to be received by SMS text message.

  • Report: This are daily, weekly, and monthly emails with detailed performance data. These are too large to be viewed by SMS text message.

Setting up notifications requires the following steps:

  • Configure the email server and sender (“from” address)

  • Add each email recipient. This includes identifying if that email destination can receive large emails (reports). Each email recipient must give permission to receive emails by replying with a 4-digit passcode, which you must enter into Network Mom.

  • Manage email notification: For each map and each email address, configure what types of email notifications it receives.

The following menu items are related to notifications:

  • Notifications -> Delete email recipient:  This deletes this email recipient and unsubscribes them from all notifications.

  • Reports -> Email Notification Configuration Report : This shows a summary report of each map, email address, and reports to which it is subscribed.
