Configure Email Recipient

To configure an email recipient select the  Notifications -> Configure Email Recipient  menu.

Network Mom LLC does not want anyone subscribed to automated email alerts without their permission. Network Mom Availability calculates and emails a deterministic 4-digit permission code to each proposed email recipient. That code must be entered into Network Mom Availability before that email address can be sent email alerts or reports. The same email address always results in the same code. It is not necessary to “keep the window open” between generating/emailing the permission codes and entering the responses. This system is designed to prevent accidental email spamming, not intentional malicious spamming.

Enter the recipient name and email address. Select whether the email address goes to a SMS/pager (alerts only) or a regular email account (reports allowed). Then click “send test email with permission code”.

This will result in an email being sent to that recipient with a 4-digit permission code and explanatory message:

The user should reply to the sender with that code. Then you reselect Notifications -> Configure Email Recipient and reenter the name, email address, email type, and permission code. Then  hit return in the permission code field . If you enter the correct code the “created new email recipient” message will appear.
