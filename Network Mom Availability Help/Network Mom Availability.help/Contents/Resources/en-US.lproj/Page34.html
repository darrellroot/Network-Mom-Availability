Manage email notifications

Select the Notifications -> Manage Email Notifications item to manage notifications.

  • Select the map

  • Select the email address

At this point the current notification setting for that map/email combination will automatically be populated. Change it as needed.

If the email address is an “alert only” email address, the “reports” options will not be available. You will need to go to  Notifications -> Configure Email Recipient  to allow reports. This will require the 4-digit permission code.
