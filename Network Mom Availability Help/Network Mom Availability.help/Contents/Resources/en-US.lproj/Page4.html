Setting up Network Mom Availability

Here are the steps to build a fully operational Network Mom Availability monitoring system.

  • Install and configure your monitoring system Macintosh.

  • Install “Network Mom Availability” from the MacOS App Store.

  • If needed, configure and test the program’s ability to automatically start after a reboot (only possible with some configurations).

  • “Maps -> New Maps” to create your maps (one map is configured by default).

  • “Monitors -> Add IPv4/IPv6 Monitor” to add monitoring targets to a map.

  • “Notifications -> Configure Email Server and Sender” to configure your “from” email address and credentials.

  • “Notifications -> Configure Email Recipients” to create potential alert and report recipients. To prevent accidental email “spamming”, a 4-digit “permission code” is emailed to reach requested recipient. You must enter this code into Network Mom Availability before automated alerts or reports can be sent to that recipient.

  • “Notifications -> Manage Email Notifications” to subscribe email recipients to alerts and reports.

  • “Maps -> Save”. While the application automatically saves data every 24 hours and before each graceful application exit, we recommending manually saving after major configuration changes.
