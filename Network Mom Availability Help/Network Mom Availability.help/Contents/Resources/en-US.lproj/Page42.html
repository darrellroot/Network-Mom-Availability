Version 1.0: February 2019

Initial Features in version 1.0 February 2019

  • Creating Maps

  • IPv4 ICMP monitor (availability and latency)

  • IPv6 ICMP monitor (availability and latency)

  • Email server settings

  • Adding and deleting email recipients, including “permission” code process.

  • Audible alerts

  • Email alerts

  • Email reports (daily, weekly)

  • Menu reports (daily, weekly)

  • Diagnostic statistics

  • Diagnostic logging

  • About, license agreement, privacy policy, credits

  • Help system

  • Export map monitor list as text

  • Saving data in core data

  • Arrange monitors in map

Known Issues in version 1.0 February 2019

  • Email attachments > 8KB not working with Yahoo Mail as the “from” address/server.

  • Running Network Mom Availability for an extended period with IPv4-only monitors may cause the IPv6 send socket to fail. Later addition of IPv6 monitors may not work until Network Mom Availability is restarted.

  • Overall map availability graphs show up in email reports but not on-screen reports.
