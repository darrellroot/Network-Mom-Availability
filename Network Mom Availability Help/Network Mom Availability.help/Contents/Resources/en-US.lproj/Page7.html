Configuring Automatic Restart

Configuring Network Mom Availability to automatically restart after a power outage is difficult and only possible with some configurations. Here are the steps to attempt it:

  • Configure the monitoring Macintosh to restart automatically after a power failure in  System Preferences -> Energy Saver . This is only possible for desktop systems.

  • In  System Preferences -> Users & Groups -> Login Options  enable automatic login. This may require only having one user account on the system.

  • In  System Preferences -> Users & Groups -> Login Items  add Network Mom Availability to the list of applications to automatically start when the user logs in.

We recommend testing that this works, but make sure to manually  Maps -> Save  in Network Mom Availability before abruptly power cycling the system. Abruptly power cycling the system is dangerous to the system. Do so at your own risk.

Enabling automatic login is a security risk. This makes it easy for anyone with physical access to the machine to obtain data. If you enable automatic login, we recommend:

  • Placing the monitoring Macintosh in a secure area, such as a manned network operation center or locked office.

  • Using a dedicated Macintosh, with dedicated (non-personal) monitoring accounts. This minimizes the data that can be stolen from the machine.

An alternative is to put your Network Mom Availability host on a UPS, although we have found that the UPS is often the most likely component to fail (especially with the “power off” button most desktop UPS’ prominently feature.

This may be one advantage of using a laptop for a monitoring host: it has an internal battery to survive power outages.
